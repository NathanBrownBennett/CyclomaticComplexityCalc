CCcalc app

CCcalc is a tool that calculates the cyclomatic complexity of your code. Simply upload your code file and the tool will do the rest.

O[N] Calc app

O[N] Calc is a tool that calculates the time complexity of your code. Upload your code file and the tool will provide an estimate of the time complexity.

Line Counter App

Line Counter is a tool that counts the number of lines in your code. Upload your code file and the tool will count the lines for you.

SuperMetric App

SuperMetric is a tool that provides various metrics about your code. Upload your code file and the tool will provide metrics such as cyclomatic complexity, line count, and time complexity.
